Github repo


Als allereerste, als je nog geen github repo hebt gemaakt doe dat dan als volgt:
Zodra je ingelogd bent op je github klik rechts bovenin op je profiel foto en klik nieuwe repository.

Download dan git bash en open het na installatie.
Type dan in:

cd Desktop
mkdir github_test
cd github_test

Ga dan terug naar github en ga naar de repo die je zojuist hebt gemaakt.

git clone (plak hier je link)

Dan kan je nu in die folder gaan en bestanden er in zetten die je naar je github repo wilt pushen (uploaden)
Als je dat gedaan hebt voer je daarna dit uit in je git bash:

git add .
git commit -m 'Mijn eerste github repo!'
git push
